# Boulder Zona

Modern, mobil-first marketing weboldal a debreceni **Boulder Zona** falmaszó terem számára. A projekt célja új vendégek bevonzása, az árlista és nyitvatartás gyors elérése, valamint a közösségi hangulat bemutatása.

## Commands

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Production build
npm run build

# Preview production build
npm run preview
```

## How we work with AI tickets

- Development is organised into sequential, numbered tickets (see `CLAUDE.MD` for the roadmap).
- Each ticket has a clear scope and definition of done.
- AI agents **must** follow `CLAUDE.MD` rules during every implementation step.
- One ticket at a time — do not modify code outside the current ticket's scope.
